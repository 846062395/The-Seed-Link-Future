# SAO Battle Simulator 计划说明
### 计划说明  
**SAO Battle Simulator会是一个注重物理交互体验，沉浸式刀剑战斗的动作游戏（魂like敌人强大类型）**
* 游戏平台：PCVR  
* 游戏类型：冷兵器战斗/魂like  
* 游戏模式：单人闯关/多人BOOS战(待定)  
* SAO特有的技能释放  
* 场景组成方式：类VrChat
* 通关竞速  
* 追求画面
* 使用原著中的交互界面
* 提供原著中相同的登陆过程
* 重要的风景场景  
* 需要：**建模/渲染/技术美术/数值系统设计**
